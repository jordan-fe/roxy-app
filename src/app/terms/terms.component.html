Politique de Confidentialité

1. Responsable du traitement

Le responsable du traitement des données collectées via ce site est Michel Jordan, joignable à l'adresse e-mail suivante : {{'jordan_michel@outlook.be'}}.

2. Données collectées

Lorsque vous utilisez le formulaire de contact, nous collectons uniquement les informations nécessaires pour vous répondre :

  • Votre nom
  • Votre adresse e-mail
  • Votre message
  • Votre numéro de téléphonne

Ces données sont transmises par e-mail et ne sont pas stockées sur ce site ou dans une base de données.

3. Finalité du traitement

Les informations recueillies via le formulaire de contact sont utilisées exclusivement pour répondre à votre demande. Une fois le message envoyé et traité, aucune donnée n'est conservée.

4. Partage des données

Vos informations ne sont transmises à aucun tiers et ne sont utilisées que dans le cadre de l’échange initié via le formulaire de contact.

5. Vos droits

Puisque les données ne sont pas stockées après l'envoi du message, nous ne conservons aucune information permettant une suppression ou une rectification ultérieure.

Cependant, si vous avez des questions sur la gestion de vos données, vous pouvez nous contacter à {{'jordan_michel@outlook.be'}}.

6. Sécurité

Nous prenons les mesures nécessaires pour assurer la confidentialité des messages envoyés via le formulaire de contact.

7. Contact

Pour toute question relative à cette politique de confidentialité, vous pouvez nous contacter à {{'jordan_michel@outlook.be'}}.
